## ADDED Requirements

### Requirement: Enhance SEO footprint via static FAQs
The bottom half of the homepage must include static structured text for SEO crawlers.

#### Scenario: Search engine crawler accesses homepage
- **WHEN** Googlebot loads the site
- **THEN** it finds semantic HTML (`<h1>`, `<h2>`) outlining the tool's use cases and FAQs.

#### Scenario: User reads the lower page
- **WHEN** the user scrolls past the main builder 
- **THEN** they see an unobtrusive, text-heavy description area explaining the tool.
